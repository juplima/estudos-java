# Estudos em Java

Este repositório reúne projetos desenvolvidos durante o aprendizado da linguagem Java,
incluindo exercícios básicos e aplicações acadêmicas com interface gráfica.

## Projetos incluídos

### Fundamentos de Java
- Uso de arrays (vetores)
- Estruturas de repetição
- Entrada e saída de dados
- Tratamento de exceções

### Java Swing
- Sistema desktop com múltiplas telas
- Telas de login e módulos administrativos
- Projeto acadêmico sem persistência de dados

## Tecnologias utilizadas
- Java
- Java Swing
- Apache NetBeans
